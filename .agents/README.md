# Noctalia Plugin Agents

Agentes curtos para acelerar o trabalho neste repositório.

- `plugin-scaffolder.md`: cria o esqueleto inicial de um plugin
- `manifest-reviewer.md`: revisa `manifest.json` contra docs e runtime
- `qml-entrypoint-auditor.md`: audita entry points QML
- `registry-publisher.md`: atualiza e valida `registry.json`
- `noctalia-docs-sync.md`: sincroniza nossa documentação interna com a doc oficial
- `local-test-preparer.md`: prepara o plugin para teste local sem executar o teste

Todos são deliberadamente pequenos e focados.
